Overview:

In this task you will be categorizing images based on their real-estate definitions.

CATEGORIZATION:

Carefully select from the available categories an appropriate classification for the image shown.

Rules and Tips:

  1. Carefully go over all available options. What might appear as an indoor -> living room might actually be a closeup -> fireplace.
  2. Example images are placed at the bottom, use these to help you make a decision.
  3. A category, like indoor, might have multiple levels, like indoor->bathroom->full bathroom or indoor->bedroom->nursury. The HIT will not allow you to progress unless you have reached the end of the path you've chosen.
  4. After selecting a classification, optional checkboxes are provided, asking "Does the image have any of the following?"
